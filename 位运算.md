# 位运算
```go
x^0       = x
x^1s      = ~x // 1s = ~0, 即全是1的二进制序列
x^(~x)    = 1s
a^b       = c => a^c = b => b^c = a
a^b^c     = a^(b^c) = (a^b)^c
```

实用:
```go
x&1 == 1 // 判断奇偶: 1是奇数, 0是偶数
x = x & (x-1) // 清除序列最后面(即最低位)的1
x & -x // 得到序列最后面(即最低位)的1
x & (~0<<n) // 将x最右边的n位清零
(x>>n) & 1 // 获取x的第n位值
x & (1<<(n-1)) // 获取x的第n位的幂值
x | (1<<n) // 仅将第n位置为1
x & (~(1<<n)) // 仅将第n位置为0
x & ((1<<n)-1) // 将x最高位至第n位(含)清零
x & (~((1<<n)-1)) // 将第n位至第0位(含)清零
```
